🌿 1. Gestor de Rutas Naturales y Senderismo en España

Tema: Naturaleza / Turismo Rural

Descripción:
Una app para explorar rutas de senderismo y naturaleza por toda España. Puedes registrar rutas, niveles de dificultad, duración, puntos de interés y opiniones de los usuarios.

Tablas:

  • Users (usuarios registrados)

  • Routes (rutas con detalles como ubicación, dificultad, duración, etc.)

  • PointsOfInterest (lugares destacados en una ruta: miradores, cascadas, etc.)

  • Comments (comentarios sobre las rutas)

  • Favorites (relación de usuarios con rutas favoritas)

Extra:

  • API pública para mostrar rutas y comentarios

  • Views para ver rutas, registrarse e iniciar sesión

  • Filtro por comunidad autónoma o nivel de dificultad

💸 2. Controlador de Finanzas Personales

Tema: Finanzas / Contabilidad Personal

Descripción:
Una aplicación web para ayudar a los usuarios a gestionar sus gastos, ingresos y presupuestos mensuales.

Tablas:

  • Users

  • Transactions (gastos o ingresos, con categoría y fecha)

  • Categories (alimentación, transporte, ocio, etc.)

  • Budgets (presupuestos por categoría o mes)

  • Goals (objetivos de ahorro o de gasto)

Extra:

  • API para consultar estadísticas mensuales

  • Dashboard con resumen financiero en las Views

🧭 3. Guía de Turismo Rural y Casas Rurales

Tema: Turismo Rural

Descripción:
Un sitio para encontrar casas rurales y actividades en zonas poco turísticas de España.

Tablas:

  • Activities (actividades en la zona: senderismo, enoturismo, etc.)

  • Bookings (reservas de usuarios)

  • Reviews (valoraciones de casas y actividades)

Extra:

  • API para apps móviles con casas y actividades

  • Views para explorar casas por provincia o tipo de actividad

📍 4. Mapa de Lugares Naturales de Interés en España

  • Ratings (valoraciones)

  • Tags (etiquetas tipo “playa”, “bosque”, “mirador”)

Extra:

  • API REST para consumir desde apps móviles o frontend

  • Filtros por tipo de lugar, localización, puntuación

📁 /models/user.js

js
module.exports = (sequelize, DataTypes) => {
  const User = sequelize.define('User', {
    name: DataTypes.STRING,
    email: { type: DataTypes.STRING, unique: true },
    password: DataTypes.STRING
  });

  User.associate = (models) => {
    User.hasMany(models.Transaction, { foreignKey: 'userId' });
    User.hasMany(models.Budget, { foreignKey: 'userId' });
    User.hasMany(models.Goal, { foreignKey: 'userId' });
    User.hasMany(models.Category, { foreignKey: 'userId' }); // opcional
  };

  return User;
};

💸 /models/transaction.js

js
module.exports = (sequelize, DataTypes) => {
  const Transaction = sequelize.define('Transaction', {
    type: DataTypes.ENUM('income', 'expense'),
    amount: DataTypes.DECIMAL(10, 2),
    description: DataTypes.STRING,
    date: DataTypes.DATEONLY
  });

  Transaction.associate = (models) => {
    Transaction.belongsTo(models.User, { foreignKey: 'userId' });
    Transaction.belongsTo(models.Category, { foreignKey: 'categoryId' });
  };

  return Transaction;
};

🏷️ /models/category.js

js
module.exports = (sequelize, DataTypes) => {
  const Category = sequelize.define('Category', {
    name: DataTypes.STRING,
    type: DataTypes.ENUM('income', 'expense')
  });

  Category.associate = (models) => {
    Category.hasMany(models.Transaction, { foreignKey: 'categoryId' });
    Category.hasMany(models.Budget, { foreignKey: 'categoryId' });
    Category.belongsTo(models.User, { foreignKey: 'userId' }); // opcional
  };

  return Category;
};

  });

    Budget.belongsTo(models.User, { foreignKey: 'userId' });
    Budget.belongsTo(models.Category, { foreignKey: 'categoryId' });

};

🔁 En tu /models/index.js

Asegúrate de que Sequelize reconozca las asociaciones:

js
Object.keys(db).forEach(modelName => {
  if (db[modelName].associate) {
    db[modelName].associate(db);
  }
});
